A GitHub theme for [Prism.js](http://prismjs.com/)

## Installation

To install the package, run the following command:

```bash
npm install --save @jongwooo/prism-theme-github
```

Or if you're using yarn:

```bash
yarn add @jongwooo/prism-theme-github
```
